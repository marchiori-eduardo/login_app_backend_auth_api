# (RASCUNHO) Login Auth API (Backend)

API RESTful desenvolvida com **Spring Boot 4** para gerenciamento de autenticação e usuários. Este projeto serve como o backend robusto para a aplicação de Login.

🔗 **Frontend:** [login-app-frontend](https://github.com/marchiori-eduardo/login-app-frontend)

## 🚀 Tecnologias

* **Java 21+** (Requisito para Spring Boot 4)
* **Spring Boot 4.0.2**
* **Spring Security** (Autenticação e Autorização)
* **JWT (JSON Web Token)** com `auth0-java-jwt`
* **Spring Data JPA**
* **Spring Web**
* **PostgreSQL** (Produção) / **H2** (Dev/Test)
* **Lombok**
* **Maven**

## ⚙️ Como Rodar

1.  **Clone o repositório:**
    ```bash
    git clone [https://github.com/marchiori-eduardo/login_app_backend_auth_api.git]
    ```

2.  **Acesse a pasta do projeto:**
    ```bash
    cd login_app_backend_auth_api
    ```

3.  **Execute o projeto:**
    ```bash
    ./mvnw spring-boot:run
    ```
4.  **API estará disponível em http://localhost:8080.**
